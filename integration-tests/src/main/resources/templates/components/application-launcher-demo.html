Documentation

Application launcher pattern

Composition only — no runtime template. Use a plain menu-toggle with the waffle icon, attached to a CSS-grid menu of icon links.

Usage

Anchor right. Standard placement is the far-right of the masthead, next to the user avatar and notifications.
