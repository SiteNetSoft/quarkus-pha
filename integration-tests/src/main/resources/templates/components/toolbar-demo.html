Documentation

Toolbar props

Parameters accepted by {|{#include components/navigation/toolbar ... /}|}. Items live in the slot — typically pf-v6-c-toolbar__group + pf-v6-c-toolbar__item.

* required
Name Type Description
* id String DOM id.
slot content Qute block Toolbar items + groups.

Usage

Pair with Table or Data list. Toolbar above a data view is the canonical pattern — search/filter on the left, page-level actions on the right.
