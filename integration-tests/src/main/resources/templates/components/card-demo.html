Documentation

Card props

Parameters accepted by {|{#include components/data-display/card ... /}|}. Inner structure (title / body / footer) is hand-rendered in the slot using PF subclasses.

* required
Name Type Description
* id String DOM id.
compact / displayLg / plain / flat / rounded / fullHeight boolean Layout modifiers — emit matching pf-m-* classes.
slot content Qute block Inner structure — typically .pf-v6-c-card__title + .pf-v6-c-card__body + .pf-v6-c-card__footer.

Usage

Pair with Gallery. The canonical card grid is Gallery layout + cards as children — auto-fill, equal width, gutters.

Clickable cards. For full-card click targets, wrap the card in an <a> or attach role="button" tabindex="0" + click handler. Ensure focus styles are visible.
