Documentation

Navigation pattern

No runtime template — the markup is plain <nav> + PF subclasses. The base classes are .pf-v6-c-nav, .pf-v6-c-nav__list, .pf-v6-c-nav__item, .pf-v6-c-nav__link, plus .pf-v6-c-nav__section + .pf-v6-c-nav__section-title for grouping.

Usage

Server-marked active item. Mark the active link with pf-m-current + aria-current="page" on the server; no client state needed for the nav rail.
