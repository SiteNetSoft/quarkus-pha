Notification Badge

PatternFly v6 notification badge component for indicating the presence and state of notifications.

Read

A notification badge in the read state indicating all notifications have been seen.

Unread

A notification badge in the unread state indicating there are new notifications to view.

Attention

A notification badge in the attention state indicating urgent or critical notifications.
