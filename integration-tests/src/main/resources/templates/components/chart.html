Charts

Server-rendered chart wrappers. The shared pha-c-chart wrapper hosts an Alpine factory (phaChart) that pipes an ECharts option object into an auto-resizing canvas with a PatternFly-aligned color theme.

Chart types

Area chart

Filled line chart for cumulative values over time.
Bar chart

Categorical comparisons — vertical or grouped.
Box plot chart

Distribution per category — quartiles, median, outliers.
Bullet chart

Actual vs. target with quantitative range bands.
Donut chart

Pie with a hole — room for a centred label.
Donut utilization chart

Single-value utilization gauge with threshold colour.
Line chart

Continuous values over time — single or multi-series.
Pie chart

Parts of a whole — use when categories are few.
Sankey chart

Flow diagram — directional volume between nodes.
Scatter chart

Correlation between two numeric variables.
Sparkline chart

Tiny inline trend chart with no axes or legend.
Stack chart

Stacked bars — parts of a whole compared across categories.
Threshold chart

Line chart with a warning + critical band overlay.
