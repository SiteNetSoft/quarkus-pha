Topology

Graph-visualisation wrappers built on Cytoscape.js. The shared pha-c-topology wrapper hosts an Alpine factory (phaTopology) that accepts a Cytoscape config — elements, layout, style — with a PatternFly-aligned default theme.

Topology demos

Custom nodes

Per-node colour, shape and badge driven by node data.
Custom edges

Dashed, thick and coloured edges from edge-level data.
Anchors

Edge endpoints anchored to compass points instead of node centre.
Selection

Single + box-select with a live readout of the current set.
Pan and zoom

Mouse + trackpad zoom with overlay buttons for fit / reset.
Context menu

Right-click reveals a per-node Alpine-driven menu — no Cytoscape extension.
Drag and drop

Move nodes; the layout is preserved across drags.
Control bar

Overlay buttons for zoom in / zoom out / fit / reset, anchored bottom-left.
Toolbar

Persistent toolbar above the canvas for filters and layout switchers.
Sidebar

Selection opens a slide-in sidebar with the node's metadata.
Layouts

Switch between grid / breadthfirst / concentric / cose without leaving the page.
Pipelines

Build → Test → Deploy pipeline laid out left-to-right with status badges.
