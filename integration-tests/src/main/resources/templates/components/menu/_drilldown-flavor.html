{! Shared live two-level drilldown used by the drilldown modifier examples. pf-m-drilled-in moves to the root + the target submenu; pf-m-current-path marks the drilled item. The content gets a fixed height because PF's JS normally measures the active level. Params: id — root DOM id (required). modifiers — extra pf-m-* classes on the root (e.g. pf-m-scrollable). style — inline style on the root; defaults to the plain-drilldown sizing. contentStyle — inline style on __content; defaults to the fixed 180px height. start — submenu key ('rollout' | 'pause') to start drilled into. long — append extra plain items so scrollable variants overflow. !}
