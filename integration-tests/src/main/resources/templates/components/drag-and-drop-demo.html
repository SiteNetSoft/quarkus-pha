Documentation

Drag and drop pattern

There's no runtime template — drag-and-drop is a pattern, not a component. The example shows the canonical shape using PF's Data list + Alpine.

The contract: each draggable element sets draggable="true" + handles @dragstart (record source), @dragover.prevent (allow drop), and @drop (splice the array).

Usage

Persist on drop. Add an hx-post="/reorder" + body containing the new order to the @drop handler — the server stores the new sequence.

Accessibility. Drag-and-drop needs a keyboard alternative — up/down arrow shortcuts when an item is focused. PF v6 itself doesn't ship this; layer it on top.
