Rectangle Selection

A rectangle selection component that lets users select multiple items by dragging a selection rectangle over them, similar to desktop OS file selection. Translated from the original MooTools RectangleSelection to vanilla JS / Alpine.js. Drag near the viewport edges to auto-scroll.

Basic

Click and drag on the background to draw a selection rectangle. Items that overlap will be highlighted. Drag to the bottom or top edge of the browser window to auto-scroll.
