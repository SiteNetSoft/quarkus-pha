Icon sizes

Icon size tokens use rems, rather than pixels. Rems are relative units that adjust font size based on a webpage's HTML document root element size. For example, if the root size is 10px, a rem size of 1.5 would be 15px.

PatternFly's default root element size is 16px. If you change this default size, note that the following tables will no longer show accurate pixel measurements (though the rem values will stay the same).

Inline icons

Inline icons must be center-aligned horizontally when placed next to text and center-aligned vertically when stacked.

Standalone icons

Occasionally, you may need to use a standalone icon that isn't aligned with any kind of text. PatternFly supports a range of icon sizes that can adapt to these use cases, including small, medium, large, x-large, 2xl, and 3xl icons. These sizes correspond to the following font sizes and tokens:

Size
Token
Example
Small (0.75rem, 12px)
--pf-t--global--icon--size--sm
Medium (0.875rem, 14px)
--pf-t--global--icon--size--md
Large (1rem, 16px)
--pf-t--global--icon--size--lg
X-large (1.375rem, 22px)
--pf-t--global--icon--size--xl
2xl (3.5rem, 56px)
--pf-t--global--icon--size--2xl
3xl (6rem, 96px)
--pf-t--global--icon--size--3xl

Medium icons are typically the most versatile size to use in a UI. Most icons in PatternFly components are medium; other sizes are used sparingly.

Icon

Server-rendered inline SVG icons wrapped in the PatternFly v6 .pf-v6-c-icon chrome. Single Qute API across four icon sets — Font Awesome Solid / Regular / Brands and PatternFly's own pficons — with PF v6's full size, status, and behavior modifiers.

Use {|{#include components/icon name="set:slug" /}|} where set is fa, fa-regular, fa-brands, or pficon. See PatternFly icons and Font Awesome 7 for the catalogs.

Inline

Here is an {#include components/icon name="fa:circle-info" inline=true /} inline icon flowing with the surrounding sentence. The inline modifier nudges the baseline to match adjacent text.

In progress

Use inProgress=true to apply PF's busy-state modifier (commonly paired with a spinner-style icon).

{#include components/icon name="fa:spinner" size="lg" inProgress=true label="Loading" /} {#include components/icon name="pficon:spinner" size="lg" inProgress=true label="Loading" /}

Accessibility

Pass label to announce the icon to screen readers (the SVG becomes role="img" with an aria-label). Omit label for purely decorative icons (the SVG is rendered aria-hidden="true").

{#include components/icon name="fa:bell" size="lg" label="Notifications (3 unread)" /} {#include components/icon name="fa:bell" size="lg" /}
