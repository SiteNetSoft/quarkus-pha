The palette

Every chart inherits the same 10-step palette via the registered patternfly ECharts theme (see /web/js/alpine/components/chart.js). Use it in series order — the first series gets the first colour, the second gets the second, and so on. The sequence is chosen so adjacent colours stay distinguishable for users with the most common forms of colour-vision deficiency.

In a chart

A bar chart with seven series picks up the first seven colours automatically — no per-series itemStyle needed.

Overriding

For semantic colour (status, severity), bypass the palette and assign per-series itemStyle.color directly. PF chart green for healthy, orange for warning, red for critical.
