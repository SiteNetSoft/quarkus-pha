Modal

A focus-trapping dialog overlaid on the page — for blocking confirmations, focused forms, or content that requires attention. Always pair with a clear close action and a real reason to block.

Documentation

Modal props

Parameters accepted by {|{#include components/feedback/modal ... /}|}. Examples here are hand-rendered to avoid slot collisions and to wire the trigger button next to the dialog.

* required
Name Type Description
* id String DOM id on the backdrop element.
* titleText String Header title (wires aria-labelledby).
size sm | md | lg Modal width.
variant danger | warning | success | info Semantic tinting + auto title icon.
slots body + footer Qute block Modal body content + footer action buttons.

Usage

Open via event. The runtime template listens for @open-modal.window with $event.detail === id. Dispatch from any button: $dispatch('open-modal', 'my-modal').

Focus management. The runtime calls PHA.trapFocus on open and PHA.releaseFocus on close — both ship with the extension.
