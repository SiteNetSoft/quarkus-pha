{! Tab — PatternFly v6 !} {! !} {! One selectable tab (role=tab) inside a tabs list. Selection is Alpine-driven: !} {! the host x-data must define an `active` variable holding the current tabKey. !} {! Pair each tab with a tab-content panel that shares the same tabKey. !} {! !} {! Parameters: !} {! tabKey (required) — this tab's value; compared against `active`. !} {! text (required) — the tab label. !} {! icon (optional) — icon name for the icons:svg resolver (e.g. "fa:users"). !} {! count (optional) — a badge count rendered after the text. !} {! disabled (optional) — render the tab disabled. !}
